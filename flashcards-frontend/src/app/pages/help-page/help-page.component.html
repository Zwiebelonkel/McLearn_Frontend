Der Leitner-Algorithmus

Der Leitner-Algorithmus ist eine Lernmethode, die auf Karteikarten basiert. Er wurde von Sebastian Leitner in den 1970er Jahren vorgeschlagen. Die Karteikarten sind in Kästen organisiert. Am Anfang sind alle Karteikarten im ersten Kasten. Wenn Sie die Antwort auf eine Karteikarte wissen, wird sie in den nächsten Kasten verschoben. Wenn Sie die Antwort nicht wissen, wird sie zurück in den ersten Kasten verschoben. Auf diese Weise wiederholen Sie die schwierigen Karteikarten häufiger als die einfachen.

Wie man die App benutzt

Die App ist eine einfache Karteikarten-App. Sie können Stapel von Karteikarten erstellen. Ein Stapel ist eine Sammlung von Karteikarten. Sie können einen Stapel bearbeiten, um Karteikarten hinzuzufügen, zu löschen oder zu ändern. Sie können einen Stapel lernen. Wenn Sie einen Stapel lernen, wird der Leitner-Algorithmus verwendet, um zu entscheiden, welche Karteikarte als nächstes angezeigt wird. Sie können auch Ihren Fortschritt für jeden Stapel sehen.
